好久没写博客了，都快忘记之前写了啥了。

这次写博客主打一个随心，近年来人有点颓废，对很多新鲜事物都没有深入去研究，技术也落后了不少，希望借写些博客来找回年轻时啥都想学的感觉。 顺便也把一些事情整块地记录一下，把原来 LogSeq 和 Notion 中有的没有碎片整理成章。

所以博客内容会很杂，希望未来的我能理解。
